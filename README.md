## About the Lab

The IBCS Lab studies how humans think, feel, decide, learn and interact with
intelligent environments. Our goal is to create systems that understand human
cognitive, emotional and behavioral states in real time and adapt accordingly.

**Coordinator:** Assoc. Prof. Gökhan İnce
📧 gokhan.ince@itu.edu.tr · 🌐 [www.gokhanince.com](http://www.gokhanince.com/gi/)
· 🎓 [Google Scholar](https://scholar.google.com/citations?user=ffrlx5sAAAAJ&hl=en)

### Core research infrastructure
- **BCI Systems** — high-density EEG for non-invasive neural interfaces & affective state estimation
- **VR / AR** — immersive environments for behavioral and emotional study (Unity3D, HTC Vive Pro Eye, Meta Quest3)
- **Signal Fusion** — EEG, EDA/GSR, ECG/heart-rate and eye-tracking for multi-dimensional user analysis
